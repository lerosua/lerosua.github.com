C++是复杂的。虽然也叫用了些日子，但对它的印象总是很模糊。比如虚函数。

别问我啥叫虚函数，也别问有啥作用，我能回答你也是在网上抄答案而已。虽然看过定义无数次，但要真正理解还是要靠自己。因此为了不让自己忘记，尝试记一下这东西。

甭管啥多态，要记住的一个功能就是，有一个子类B,将之转化成其基类（A)指针时，也能方便地引用到B类的函数。说得有点绕，代码解释吧： 




#include 
using namespace std;



class A{

	public:

		virtual void run(){

			cout << "A->run()" << endl;

		}

};



class B:public A{

	public:

		void run(){

			cout << "B->run()" << endl;

		}

};



void play(A* p)

{
	p->run();

}
int main(int argc, char *argv[])

{
	B* p = new B();

	play(p);



	return 0;

}
