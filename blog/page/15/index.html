Ejabberd服务器transport网关j2j扩展折腾记

| Comments

ejabberd服务器，xmpp协议服务器。
今天我要折腾的就是给这个服务器加个j2j扩展。那啥是j2j呢，就是jabber to jabber. 本来大家都是用xmpp协议，只要服务器开通了s2s特性，就能互联。比如以前的jabber.org连gtalk，说到这，又要责怪一下google放弃xmpp协议搞hangouts啥的了，你搞就搞，关闭xmpp协议干鸟啊，少了互联特性，一点也不geek了。
好了，说回来，这个j2j的transport就是为了那些不开放s2s，各自为政的xmpp协议服务的。比如新浪微博，它私信用的是xmpp，但不支持s2s，所以只要有j2j扩展，就可以把gtalk连到新浪微博上, yes i can.

操作步骤参考wifi : http://wiki.jrudevels.org/Eng:J2J:AdminGuide

首先安装好依赖：python > 2.3, python-mysqldb, Twisted 等等。

下载源码:

svn co svn://svn.jrudevels.org/j2j/trunk j2j/

设置数据库：
可以使用postgresql或mysql，　我这里使用mysql

　mysql -h localhost -u root -p -e "CREATE DATABASE j2j;"

mysql -h localhost -u root -p -e "GRANT SELECT, INSERT, UPDATE, DELETE, CREATE, DROP ON j2j.* TO 'j2j'@'localhost' IDENTIFIED BY 'yourpassword';"

然后这句，说是导入数据库结构。

mysql -h localhost -u root -p -e "use j2j; \. mysql.schema"

至此数据库设置完毕。

ejabberd服务器配置:
英文文档里少了这一步，害我折腾了好久，在ejabberd的配置eajbberd.cfg里要把transport的配置打开，一般它的位置是 /opt/ejabberd-2.1.11/conf/eajbberd.cfg

  {8888, ejabberd_service, [
                            {access, all},
                    {shaper_rule, fast},
                    {ip, {127, 0, 0, 1}},
                    {hosts, ["j2j.example.org"],
                     [{password, "xxxxx"}]
                            }
                           ]},

当然example.org是你ejabberd的服务器域名，并且把j2j.example.org的子域名设置好。编辑完配置则重启服务。

网关配置:
回到j2j的源码目录，编辑j2j.conf

cat j2j.conf
[component]
JID=j2j.example.org
Host=127.0.0.1
Port=8888
Password=xxxxx
Send_probes=True

[process]
Pid=/var/run/j2jt/j2jt.pid ; pid file

[database]
Host=localhost
Type=mysql ;or mysql
User=j2j
Name=j2j
Password=数据库密码
Prefix=j2j_
mysql_ping_period=2 ; ping period for mysql in hours. 0 for none

[admins]
List=lerosua@gmail.com,admin2@example.net,admin3@some.net
Registrations_notify=True

[debug]
logfile=/var/log/j2j/j2j.log
registrations=yes
logins=yes

xml_logging=/var/log/j2j/xml.log
component_xml=no
clients_xml=no
clients_jids_to_log=All ;or user1@server,user2@server

主要配置好component这一段问题就不大了。

最后运行:

sudo ./main.py

用root权限去跑。

修复gnome-shell默认主题中图片查看器等与其它程序不一致的问题

| Comments

gnome-shell的默认主题是Adwaita，从发布第一个版本开始，eye of gnome, gnome-mplayer,totem等这几个程序的界面就跟其它Gtk+程序不一样，其它程序都是选好的亮色，偏偏这几个程序是暗色系的。换了好几个主题都是如此。

这次重新安装系统，终于在网上找到一知半解的解答：http://askubuntu.com/questions/132556/why-totem-eog-have-dark-toolbars-with-default-gnome-shell-theme-adwaita

完全是主题作怪。把 /usr/share/themes/Adwaita/gtk-3.0/ gtk-dark.css 这个文件删除即可。

然而metacity的主题也有一样的问题…

智能手机之殇

| Comments

首先得承认，这篇是标题党。用那个殇字纯粹是为了好听，殇代表死亡，而要提的这个问题最多是智能手机的硬伤而已。对的，且不管智能手机智能不智能。如今它能代替电脑，甚至许多电脑都做不到的事情。各类功能，娱乐app层出不穷。但是，通常，半天就要没电了。

电子产品都有这类硬伤，听说google glass也坚持不了半天，那谈何整天跟在身边的穿戴设备啊。

如今智能手机厂商尚不能完美解决的续航问题，结果，它们还将之恶化，即将电池内置不可更换。这还叫啥移动手机啊，一天中有一半时间是要连接到电源线里的，像个整天打点滴的病人一般。

电池技术的进步可有两种，一种是增大容量，当然是不加大面积的情况下。一种是减小充电时间，最好是秒充。

然则目前技术无法猛进的情况下，简单的解决方案也是两种，一是仍然增大容量，尽量加大手机面积，压缩其它配件的空间。二是，换电池吧！

第一种方案当然大家都在做，但手机始终有个大小限制，但一些厂商有回归大哥大那种砖头的趋势。
第二种方案却无人问津，越来越多厂商把电池做成不可更换的。

为什么不在换电池这方案上做优化呢。增加内部电池，在换电池的过程中不用关机。优化换电池的方式，不用开后盖之类，加个滑动机关，一开即可将电池优雅地滑出，然后将第二块电池优雅地推进去。整个过程不需要十秒。手机电量立马原地复活。这样在郊外备人十块八块电池，真是永不掉电啊。

一个个只会跟在苹果后面学不可更换电池，Android请有Android的样子好不好，本来就比不了iphone的省电节能了。

工作：上周连续工作直到周三，工作有点正规化，让同事们通过一个easybug.net平台向我提交bug，然后我一一修改。有记录，有反馈，多少摆脱了一点小作坊模式的口头交流反馈。然后经理们飞去北京，其余员工们直接开放五一假，算起来有七天。一开始还想着要不要去玩，然后在家呆了两三天，果然还是宅下去。明后天回趟老家，假期也就这样了吧。

放假：早已经没有什么雄心壮志了，于是找电视剧来打发时间，连续看了不同人马演绎的『鹿鼎记』，高中的时候我是看过小说的，但是全忘了，现在的印象基本上都是从电视剧里来的。陈小春版剧情应该贴近小说，但实在有许多不能自圆的地方。张卫键版主要是看美女，虽然舒琪只是串一下，但找的演员基本都能达到美女的要求。对『鹿鼎记』里的世界观最不认同的是少林寺，当年的少林寺可是反清复明第一基地啊，所以才有后来火烧少林，五祖出走之事。可『鹿鼎记』里的少林寺随便一个圣旨就让韦小宝当了师叔，还屁颠屁颠地跟着去保卫顺治。可谓满清第一走狗是也。

TVB的剧集也是烂出境界，那个『神探高伦布』不破案只发疯就算了，『金枝欲孽II』其实只不过是拍了部『大太监』，如果再拍一部『大嬷嬷』罢了。或者直接叫『紫禁城里宫女的那些事儿』。

其它：人类科技水平，还真是继续低下。其实这几十年来什么也没解决，就是发展了娱乐而已。在醉生梦死这一方面，成就倒进展卓越。却越发令人空虚寂寞。我看着手机里满满的通讯录，却找不到一个可以聊天的人。

突然很厌倦Android手机，喜欢上黑莓的那种操作方式，全手势操作，并且想要message hub那样的东西，集成邮件，短信，twitter等。但是目前所有的手机都有续航的问题，一天一充真是反人类的大倒退。我现在这部Meizu MX用了已经一年多，看看能不能撑到三五七年的，等到电池真有所越进的时候。

自家的VPS前几天突然出了点问题，恢复之后导致我的xmpp服务器一直不能正常使用。重新部署了好几次都仍错误，找不到原因。于是聊天室房间再次告挂。实在是没运气。

下周星期四五才上班。四月份基本就如此过去了，没有可追忆的东西。

多难之邦

| Comments

实在是找不词语表达对雅安地震的心情。『雅安不哭』，『雅安加油』这样的词语不觉于网络。或者把网页设置成黑白的，禁止玩游戏一两天？还是捐钱？

相比起五年前的汶川地震，我们要麻木的多了。或者清醒得更多了。我们明白一切未变，建筑仍然未有质量，民生不会有何改善，改善的是官员的演技。

古时一个地方受灾，全民捐款，地方官员从中取利，因而更加不修公益，导致越发多灾，越多灾就越多财。而如今进步了，不但不修防灾害的公益，甚至修个能引起灾害的事物『如豆腐楼，如水坝？』。

所以捐款这事，这个地方没有可靠的组织保证款项的正当流向。大家也都明白了，于是又发明逼捐这种事情。政府几万亿地印钞，几万亿地援交，一讲到民生却又在哭穷，没见过如此不要脸的。

但天朝这块土地的人真是很有适应性。蒙古人来了，于是把成吉思汗当成自己的英雄，满族人来了，可以当康乾当盛世。日本人来是来了，没来成，后来没心没肺的人来了。从此这块土地没心没肺了。

R.I.P 所有中国人
