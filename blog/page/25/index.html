你好敏感啊

| Comments

頭先系推上得知，廣州市民又一次系人民公園集會，反對朝庭嘅推普反粵。但系例次引來著更大規模嘅公安。甚至出去到大巴。而且例兩次事件全都系媒體上全面封殺，尋吾到一滴消息。
我地廣州市民都經已承認普通話嘅通行了。系學校嘟賓個唔系講普通話啊。連我地私下裏交流講嘅話都要幹涉。殺著我地白話，同系滅著我地文化有什麽區別。所以廣州市民大家都系自發性嘅起來示威。是可忍也，孰不可忍也。

例地本來是廣州市民想要表達嘅想法。但系市民有咩途徑去表達呢。冇！所以才有甘多市民上街表達意願。不過系朝庭嘟看來，你們唔系表達意願，系嘟示威。黃子華棟篤笑裏都講過啦，一個威權，面對弱小嘅一方，系唔會讓步嘅。弱小嘅一方無論提出咩吔要求。單單系提要求例個動作經已系反抗行為。所以朝庭唔會理你系提粵語，或者更直接地提民主，例地要求無一唔系變成提你老母嘅。所以你無論提咩要求，一有大批民眾聚集，經已觸發著朝庭嘅敏感系統，高潮般射出大量公安。此即朕射你有罪。

其實我嘅粵語講得麻麻地，講心個句，就算身系廣州，我都未必會一同上街。系啊，我系幾怕事咖。所以表白著良心先。
都講著甘多了，唔掙埋講埋例句了，軍隊國家化啊。

vim技巧之取词翻译III

| Comments

以前写过vim里用sdcv取词的一系列文章。

按一个快捷键能调出窗口来显示sdcv的翻译结果，但它不能方便地调用关闭。也就是用同一个快捷键开关窗口。如今此vim脚本有了进化，加了如此功能。

全部脚本如下： 

let s:w_mywin=0

function! Mydict()
if s:w_mywin
let s:w_mywin=0
set nu!
close
file
return
endif
let s:w_mywin=1
let expl=system(‘sdcv -n ’ .
\ expand(””))
windo if
\ expand(“%”)==”diCt.css” |
\ q!|endif
botright vertical 25split diCt.css
setlocal buftype=nofile bufhidden=hide noswapfile
set nu!
1s/^/\=expl/
1
endfunction
nmap :call Mydict() 将之保存为sdcv.vim,放到~/.vim/plugin即可。

再次无题

| Comments

一段时间都没有写博客了。甚至有些遗忘这里。感觉这里一直是我理想的阵地。超越现实的精神世界。

而前一阵跟老婆聊天，她说我转变了。从以前的理想主义转变成了现实主义。因为她以前问我在干什么的时候，我似乎都在写程序，而现在问我在干什么，我答不上来。似乎不是看电影就是上网。上网上网，无尽的上网。以前的我迷失在北京了吗。

网络上有太多的新闻，现实的消息，然而不能增进我的思考，也只能加剧人的愤怒。

为了打发地铁上的时间，我买了一个电子书设备，就是电子墨水的那种。看txt还行，pdf刷新有点问题。在看一本禁书，《远×华×案×内幕》。里面讲的官场黑暗，全党溃败。当然里面人说的话不能尽信。也只能当小说看。因为你当现实看了，那会扼杀希望的。

博客里是不应该发这些不和谐的言论的，然而在twitter里就可以。在twitter出现的时候，总认为博客才是正道。现在看起来，twitter才是能自由畅言的地方。然而twitter默认是在墙外的。我们这么争取言论自由，不正是因为这是我们最缺少的东西嘛。而党国一再强调要和谐，不正是因为这个国家一度地不和谐嘛，而常摆在嘴边的以法治国，不正是说明国家经常就不这么干嘛。经常说反腐，不正因为腐败广泛而深入嘛。

为什么中国人命运就如此坎坷呢。我发这些唠叨又有什么用呢。

24小时第八季完结

| Comments

一直都在追的美剧。《24》，主要讲米国ctu探员jack在一天24小时内拯救美国，粉碎恐怖分子的事情。将剧情压缩至一天内，张力十足。这八天中，多次破解核弹危机，病毒危机，以及米国总统阴谋。甚至连天朝大使馆也没放过，呵呵。

以下为剧透：

第八季一直围绕着一个所谓和平协议开展的。看到23集的时候，我真替泰勒总统感到委曲。事实其实是，泰勒总统一直想打麻将，好不容易约齐了两个雀友。irk共和国总统和俄国总理。irk总统也是雀友，但问题就在于俄国总理不是。他非常喜欢打麻将。但总不好当面拒绝米国吧。人家说到底也是世界第一强国，总不能不给面子吧。于是搞个阴谋把irk总统给干掉了，这回三缺一，打不成了吧，正乐着。谁知道irk总统的老婆也是个麻将高手，在泰勒的劝说下顶了她老公的角色。好嘛，人齐了，准备开台吧。却在这时候俄国总理杀死irk总统的事让新接任的她老婆知道了。新irk总统就不愿意和杀夫仇人一起打牌了。泰勒总统只能软硬兼施，好不容易让这两人坐在麻将桌上。可是看着桌上的人，不就打个牌嘛，左一个哄，右一个骗，威逼利诱全用上了。泰勒总统想着想着，我咋那么二哩。这牌打起来没意思。于是半途自个走了。

当然24小时结束了，我会想念小强的。当然也想念chole。小强最后一句话，也挺感动的，说没想到chole会是一直都以来支持他的人。在24小时里，除了小强不死这个常量外，任何人都可能随时牺牲，任何探员都有可能内奸。另一个常量就是chole永远是小强坚持的后盾。他们里应外合，合作无间，所向无敌。

24小时不会再有第九季了，我都怀疑，以后能否再看到如此紧张刺激的美剧了。i will alway miss you.

20年后的生活的一些片断

| Comments

参加 通用汽车连线2030的一个活动的文章(http://www.gmexpo2010.com)，活动http://blogpk.gmexpo2010.com

1990年，每个家庭几乎都能拥有一辆自行车，街道上都是飞驰的二轮。出租司机们还在开着三轮。现在，一部分家庭都有轿车。相信20年后，轿车应该是每个家庭的基本配置。收费站应该全部消失了。那时的年青人能带一把面包就从广州直接开到北京不用加油，对他们说起这一路上曾经有过N个收费站的话，他们会像听到神话一样以为我们在开玩笑。而一部分家庭都开始标配私人飞机。当然带来不好的影响，肯定是撞机事故的增加。

然而网络科技方面的预测则是最困难的。因为这是变化最显著的地方。一定会出现我们现在想也想不到的产品。但从当年流行BP机，大哥大的趋势来看。20年后手机已然是淘汰的产品。想到这里，或许可以把不用的手机保存起来，到时也是古董一个啊，也是历史的见证啊。趋势已经很明显，笔记本和手机二合为一，浓缩成更小的体积，而拥有更强大的计算功能。或许搭载了更多适应人体工学的人工智能算法，比如无障碍沟通。再也不需要学习别的语言，无论是打电话给法国的一个客户，无需要我会法语或对方会汉语。手机会自动帮双方转换，完全无缝。而且网络无处不在。在外度假，可以使用手机关掉家里的电灯，这再也不是少数极客所做的事情，而且大部分普通人所能理解的最常用的功能。就像你和朋友在酒吧聊着天，提前用手机给自己的车子发送指令，让它从家里开来此处接你们一样，是很自然的行为。而很geek的事情或许就是调用自己的私人卫星来给自己拍个大头照了吧。

出国旅行已经像是出省一样方便。您要考虑的是怎样赚钱上月亮上转一转。是的，太空旅游开发应该是新兴发展的旅游业。像我们的老爸老妈一辈子没出过省，出过国一样被视为out，老土一样。我也会因此而被视为老土，out。因为我在听《月亮之上》，而人家是真正在月亮之上。也像一些上一代搞不清楚电脑的键盘是干嘛用的，点击一些键位会出来一些奇怪的东西。或许我在儿子的电脑面前也是如此不知所措。因为我会完全找不到键盘在哪里。而且在房间里乱动一会，墙壁前的3D屏幕会现在让我登录的命令。可我完全不知道如何输入，以及如何触发开关机的动作。或许也完全不知道儿子已经用他的眼膜指纹锁住了电脑。在我看来，儿子是经常一个人在房间里对着空气，哭笑怒骂或指手划脚。或许我就理解了现在有些父母把他们儿子的网络行为视为网瘾，而考虑把他们送去电击的冲动。

以上只是所能想到的一些小的片断。那时候，应该是二零后横行天下了。八零后都被退出江湖三十年了吧。

一周动态

| Comments

哗一下，一周过去。这是进入新单位的第一周。还算顺利。

虽然适应新环境总要一段适应时期，但好在有wind和yetist的帮助。当然还有jyf1987, 他们三人在我这过去的一个半月中给予了无数的支持与关注，让我得以顺利在北京暂时安顿。在这里先说声谢谢。出门靠朋友也是靠谱的。

上班的时间总是过得飞快，一周下来基本上只记得如何在公交地铁上挤。每次在地铁入口一大堆人中，拥挤着等待入站，自己随着人流只能五秒一次地挪着小碎步。都无不提醒着蚁民的概念为何。由于上班的路上时间加长，不少人都拿出手机，psp go，mp3来打发时间。也或许只有这段时间才能让人看一下书。我也在这时间拿出手机看电子书。前段时间刚看完唐德刚的《满清七十年》，获益良多。一直想酝酿出一些读后感，但话到手头又打不出来，不知道是不是表达能力又下降了。又或者是发表的只是前人牙慧，不耻于出手。但这绝对是值得一读的书。要看的话，找台版。大陆版的被太监过的。呃，关于这本书，迟点再专门讨论一下。

再谈点别的，由于环境的变更，坐息时间的调整。我此时还在适应中。而且也要尽快融入公司的项目中。上两周都没有动开源项目。当然也有我懒的时候，懒的人是经常有借口的。希望我能稍微更改一点。尽快找到调和开源项目的时间，当然不像以前那么有时间。现在一天三四个小时都花在路上了。

目前在做的是gmchess和pidgin的整合。效果就是通过pidgin的一个插件，能让pidgin上的好友直接在gmchess 上下棋。这大概就是一直想要的gmchess网络功能吧。基本上是我能想到而又能最快做到的方案。希望尽快能出个demo版本。
