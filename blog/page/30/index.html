随便说说

| Comments

8月一号，电信的套餐终于转成了上网版，也就是有400M流量的。中国电信客服很喜欢关爱用户，经常会打电话来问用户，比如问为什么不多用打多点电话，上多点网啊。我说过很多次，我的那个电信号码完全是电信营业厅流氓捆绑宽带时一起开通的。送的一部电阻屏的“智能手机”，落后时代五年。到后来我才搞明白，开的套餐是聊天版，120M流量，360分钟通话，管我用不用得了这么多，每月在银行卡照扣80￥。我非常非常之不爽。也后悔当时一时冲动为了带宽去开通。后来公司配了部4s当开发机，正好是电信版。用上了真正的智能机，流量当然多多不够用了。去研究电信的套餐，才明了同样价格还有个上网版套餐。也就是400M流量的。于是打10000号去客服让换。打了两次电话，结果返回的结果是不同的，一个说能转，一个死活说不能转。于是我让能转的帮转，投诉那个不能转的。
今天再去查套餐，果然转成了上网版，这电信，管理真混乱。当国外在建设G为单位的带宽时，天朝这还在为争多几十k而计算。那些所谓的砖家怎么有老脸提什么大国崛起啊。

然后新华社今天的一篇文章，归纳出新时代的黑五类，不禁让人联想，共同富裕的第二步将要执行似的，就是把不富裕的全都消灭掉，从而实现共同富裕。未来的形势不容乐观啊。

前阵子参加的同学聚餐，才得知某同学和某同学终于也分手了。大学时代我们班自产自销了三对情侣。感觉这比例也挺高的吧，他们分别于，毕业前分手，毕业就分手，毕业后分手。毕业后分手那对，其实我挺看好他们的，或许终究是熬不过异地相思吧。也许不到老，你也不知道，谁是真正会陪你终生的人。

下班回家的路上，仍旧看到了一个人硊在地上拼命地磕头，口里喊着，谢谢大家。旁边摆着一个睡着的老妇人。我走过的时候瞟了地上的纸两眼，大约是他是个司机，出了个车祸赔了钱，现在老母亲病重无钱医治什么的。先不论真假，但是我已经心安理得地把结论推给政府，要怨怨政府的不作为吧。生在天朝都是不幸人，只是有些人更加不幸。

Git服务器搭建

| Comments

因需要私人项目，需要跟朋友协助，因此需要代码管理仓库。github本来应该很好的，但我暂时没美刀支付渠道，曾想过dropbox+git本地管理，但发现dropbox已经禁用这种使用方式了。后来找了个bitbucket.org的，可支持私有项目，只是协助成员限制在五个人以内，这也没问题，但后来发现在天朝的烂网络下，太慢了。最后老犁借了个vps给我，让我在上面自己建一个。于是有了折腾。但发现其实还是非常简单的。

网上已经非常多的教程了。这里就我也啰嗦再贴一次，主要参考了

Git服务器Gitosis安装设置 , 使用gitosis的来配置管理的git服务器端

1. 准备环境：

老犁的vps装的是ubuntu，因此用apt-get安装以下工具 

sudo apt-get install openssh-server openssh-client git-core python-setuptools
然后建个git的用户 
sudo useradd git
sudo passwd git
创建git仓库存储目录 
sudo mkdir /home/git/repositories
设置git仓库权限 
sudo chown git:git /home/git/repositories
sudo chmod 755 /home/git/repositories
初始化全局设置 
git config --global user.name "git"
git config --global user.email "git@server"
2.安装git服务器

这里选用的是gitosis，教程上面全是让git clone一个出来的，这里我其实好奇怪，为啥不用apt-get直接装库里有的呢，我自己就试了下安装，但发现它在下一步gitosis-init的时候出错，难道仓库里的gitosis坏了没人管啊，于是我也只有去clone一个出来，但网上 git clone git://eagain.net/gitosis.git 这个地址似乎失效了，还好找到了替代。 

git clone https://github.com/res0nat0r/gitosis.git
安装需要用到python-setuptools 
cd gitosis
sudo python setup.py install
3.配置git服务器

身边linuxer，接触到服务器这方面，多少也早就有自己的ssh key了吧，没有的话ssh-keygen -t rsa自己生成一个，这里不细说。把自己的公钥~/.ssh/id_rsa.pub 上传到服务器。 

scp ~/.ssh/id_rsa.pub 你的名字@你的服务器:/tmp
然后用公钥去配置服务器 
 sudo -H -u git gitosis-init < /tmp/id_rsa.pub
Initialized empty Git repository in /home/repositories/gitosis-admin.git/
Reinitialized existing Git repository in /home/repositories/gitosis-admin.git/
后面两行是输出。(命令中的小于号似乎显示不了…)

然后是设置post-update脚本权限，其实我看这的权限本来已经是755了，但网上都有这一步，是保险吧。 

chmod 755 /home/git/repositories/gitosis-admin.git/hooks/post-update
至此服务器方面的操作完成。

4.管理git服务器

git服务器的一个好处是它可以通过git回客户端来配置管理，现在回到自己的客户端，也就是上传了公钥的那台机器，因为这里有能解它的私钥啊。
clone出git服务器的配置。 

git@你的服务器:gitosis-admin.git
取回来后进到gitosis-admin目录，会看到目录结构很简单，一个gitosis.conf配置文件，和一个keydir目录，里面放着之前导入的那个公钥。
配置文件类似 
cat gitosis.conf 
[gitosis]

[group gitosis-admin]
writable = gitosis-admin
members = leros@MacBook-Pro.local 这里有个建议，如无必要不要去修改管理者的密钥名啥了，之前我试过改，然后就导致 ERROR:gitosis.serve.main:Repository read access denied 的错误。另外，如果members里的名称和keydir里的公钥文件名不一致也会导致这个错误，所谓一致是指 members里是xxx，则公钥文件名y对应为xxx.pub

5.添加新项目及新加协助成员

那么，现在我们差不多可以正式工作了，建立一个新项目，app-test,邀请成员lerosuz加入协助。那么我们就必须在服务器配置文件里加入 

cat gitosis.conf 
[gitosis]

[group gitosis-admin]
writable = gitosis-admin
members = leros@MacBook-Pro.local
[group cyclone-team]
writable = app-test
members = leros@MacBook-Pro.local lerosuz 如上所示，我们新建了一个组叫cyclone-team，组员是管理员leros他新成员lerosuz，管理员获取到lerosuz的公钥,并将之命名为lerosuz.pub，加入到keydir中，然后提交修改至服务器，于是就会生效了。命令大概是： 

git add keydir/lerosuz.pub
git add gitosis.conf
git ci -m "添加lerosuz协助"
git push
以上是设置了新项目的读写权限。现在可以正式建立新项目了。
还是在笔记本终端中： 
mkdir /tmp/app-test
cd /tmp/app-test/
git init
touch testfile
git remote add origin git@你的服务器:app-test.git
git add testfile
git ci -m "inital"
git push origin master:refs/heads/master
完成以上命令，则项目就会传到服务器上，新成员lerosuz就可以clone项目下来协助了。 
git clone git@你的服务器:app-test.git
当然，没有加进公钥的人是无法clone下来的。

PS.以上的git命令我用的是简写，请确保你的主目录下的.gitconfig有以下内容: 

 cat ~/.gitconfig 
[alias]
	st = status
	ci = commit
	co = checkout
	br = branch
	dc = dcommit
	rb = rebase
enjoy~

时事：周周有大事，事事步惊云。七一香港游行，将近五十万人上街，这，就是真正的民心。今日西藏，明日香港。香港会否再次沦陷，看香港人民的决心及不可低估party的手段了。天津蓟县一场大火，突现出无信仰天朝悲催的价值观，竟然喊出了结了账才能走，且不论是否谣言，但我是信了。然而加入人祸因素的火在结束后，苦难才刚刚开始，一场大火百人以上，却只有十个死亡名单，意味着其它人只能永久失踪吗？群众连祭奠逝者的权利也要被剥夺，出去大批便衣干扰威胁。哎，我们这一辈人，生于计划，死于维稳。你得拿到准生证才能出生，你得在稳定死亡人数之中才准死，都是有限额的啊。还真是阎罗王啊。

视频：好像没看Ted，看了部旧电影『战争之王』，影片最后指出世界最大的军火贩子正是那五大常任理事国。人类，真是种可怕的种族。

社交：今天找两个旧同学唱K续旧。想不到小辟已经结婚，老婆已经七个月了。想当年只是三个人唱K，如果各自都带着女朋友，老婆。看来下一次，要带着小孩了。岁月增长，风华不再啊。

其它： 今天也终于帮公司买了iphone 4S。白色电信版。虽然之前见过iphone, 但也只有拿到属于自己的那台，才真正会有体验它的感觉，感觉到精致。或者只是新手机的感觉，又或者果粉的错觉。拍照真心快得多了。公司配机，使用权归我，等报销。

运程：★★★

期待：flymeOS会在12号发布，希望是个比较完整的版本，4.0的android，能用上gtalk视频版了吧。可惜我现在粉想4.1的google now,魅族动作太慢了啊。

周记－天

前言：写点周记吧，要不然，总觉得时间过去了，什么没有剩下的感觉。主要是前阵子工作的事情一团糟，未来不明确的状态导致人身心疲惫。

技术：预计重新复习一下算法，周一的时候看了下《数据结构与算法C++》的PDF，仍然是开头很努力地看，到后面阶段变成了随意翻页。又再学习了快速排序算法。下周要找时间继续复习这方面。然后这两天也复习了Cocoa touch的一些零散的知识。以后要经常翻阅CocoaChina里的帖子，吸引经验。放假还有两天，得做好项目App评估。虽然是很烦。It is a hassle.

语言：没想好怎么学日语，又或者要不要学，很矛盾。关键是英文也没学好。不过还是准备了一个本子，把看文档不认识的英文记下来，新公司有个混血儿老外，有空要找机会跟他练练口语，不开口真的学不了会的啊。

视频：TED看了两个左右，有个是《迷失》的编剧在说设计吧，有字幕也没明白在说啥，还有个介绍了极端天气环境下的物理研究，好吧，已经忘了内容了…

《心战》继续追着，内有文艺气息，换句话说就是剧情太闷，好在杀手面目开始铺开，开始有戏看了。

动漫上《黑子的篮球》看了第11集，全场距离射三分，也只能感叹当年《灌篮高手》太强势了，如今的篮球戏只能另辟蹊径，玩超能力啥的了。《猎人》动画继续连载，但搜狐似乎只预定了五十集，极有可能也TV版也只出到旅团篇完结，连GI篇都未必会出完，蚂蚁篇更是遥远的彼岸啊。

历史：地铁上的时间仍然是看书的好时机，开看《1901年一个帝国的背影》。看完了戊戌变法篇，正在看义和团篇。仔细观看历史的细节，更有利于发现历史的真相。戊戌变法学的是日本，想搞君主立宪。可首先条件就不成立，光绪根本没权力。其实康梁一派也纯粹理想主义者，做事天真兼天马行空。比如想架空帝国官员，给以虚名而剥其实权，怎么可能不受庞大的官僚系统的反对哩。大概在整个中国没个明白人吧，官本位文化毒害太深了。另外唐德刚的晚清七十年里也有提及变法，语之甚详，里面更完整叙述了康圣人的人格特征，自信，狂妄。总之，令人扼腕，变法失败是必然的。然后义和团篇才是真正好看的历史，才是剖析中国人性的片断。这段混乱的历史，只是六十年后那次狂欢的预演。我只是感慨中国人的脑袋被洗来洗去，从来没有干净的时候，能装得下独立的思想。

 

社交： 主动跟公司的混血儿帅哥搭讪了几句，人家普通话好得很。推上这星期算话痨了，其它环境还是沉默得多。

运程： 估计走势一般吧，一星。

期待：下周一魅族的flyme发布，MX应该能刷上4.0的系统了吧。

香港一日

| Comments

因缘际会，终于有理由有借口去香港一趟，谈不上旅游，也不是公干，是去参加GNOME Asia 2012会议。

作为一个广东人士，从小看TVB && ATV 电视及香港电影长大的，一直都想看看这个即陌生又熟悉的地方。虽然香港已经回归，但大陆人士去香港仍然是有所限制，我办的那个通行证只能一年去两次，每次停留不得超过七天，可人家英国护照可是随便停留三两个月的吧。anyway，总之可以去所谓的资本主义世界见识一下了。

上星期五下午，请好了假，收拾好了行装。从广州出来，来到东站，直搭和谐号到深圳，一个小时来到罗湖，跟着指示牌，来到过关口，人不是很多，排完了队盖好章，算是出关，然后再排队盖章，于是算是在香港境内了。中国移动的信号也不见了，开了漫游服务，转到了CSL服务商。这里要提醒一下，首次出关没经验，在罗湖关口看见港币兑换所，就直接去换了（1000人民币况11xx港币，之后在香港才发现兑换的地方很多，而且多数是（1000人民币况12xx港币）。于是发现亏了。

来到香港罗湖关口，买了张八达通，再坐地铁到市区。定的酒店在旺角，后来发现在砵兰街附近。香港地铁都比较旧啊，发现还有头等位子，是需要另外再刷一次卡的。坐了差不多半小时，到达旺角。这时天色也渐晚，终于是身处于灯红酒绿的资本主义世界了。

满街的招牌真的是看得人眼花缭乱的。

本来好不容易肉身翻墙一次，当然要体验一下无墙的浏览体验了。说真的，还真的是很容易就找到wifi信号的，可惜能连上都没有账号密码。在旺角乱逛，找了半天没有网吧… 然后去KFC吃了个晚餐，KFC上的WIFI也是PCCW这些，虽说电讯盈科（PCCW）提供访港人士24小时免费WIFI，但我的手机连不上它的网络获取不了账号密码。只能坐在KFC店里，看着往往来来的人流，感叹街道之小，人流之旺了。之后来到了预定的酒店，因为是朋友预定的，酒店不认我，让我等朋友来了才能入住，shit了个大发。还好酒店外面有一片小花园，我坐在那里等。真正感受一下旺中带静，一片繁华商业之地还能空出一大块地皮供行人休息。

之后朋友过来，入住酒店，才明白啥叫香港的寸土是金。小店的房间摆了两张床就满了，不好意思，第一次见如此精致的房间。

进入房间之后终于可以使用酒店的wifi了，也终于可以不开VPN就直刷推特了。然后打开电视，晚间新闻时间，各路人士都在关注李旺阳事件，并称星期天会组织游行。第二天苹果日报的头版也是关注李旺阳事件的。我觉得，香港已经是中国最后一块良心地了。

第二天去到茶餐厅吃早餐。门口位子坐了几个浓妆艳抹，花技招展的女子，隔壁位子则是两个中年男人，每人桌上都摆放着两部手机。以此架式看来，这就是『马夫』和性工作者了吧。香港黑帮电影经常出现的标志啊。不过这几个性工作者脸上倦容尽现，有一个还说昨晚只睡了两个小时，由此可看出她们工作艰辛啊。

这个砵兰街是出名的红灯区，以前在香港电影中经常听到『砵兰街大家姐』啥的，黑帮必提之地。

之后的行程回归主流，参加GNOME大会，会场在城市大学。关于会议内容就不在此详述。正巧城大另一个活动也进行中，我只在这活动结束后才知晓，正是『沉默的力量』刘霞摄影作品展。

晚上的时候我自个走到星光大道上游走，那里的夜景真的是很美。尤其是晚上专为游人准备的一个节目，几栋大厦射出的灯光直冲云霄，照亮整个夜空，真的是震憾斗府。对面中环的高楼大厦，幻彩霓虹，美不胜收。 这张照亮天空的照片没拍好，手机能力有限，但这场景也只有亲眼目睹才好看。 之后我就从尖沙嘴直接走回了旺角。途中还见到了ICAC，廉政公署。我相信，大陆迟早也能会有的。

人生总归是有希望的，昨晚看『心战』有句台词说得好，不能期望天天天晴，但总能肯定，总不会天天都是下雨的。
