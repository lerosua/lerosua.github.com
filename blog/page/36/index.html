GMLive 0.22.1发布

| Comments

GMLive 0.22.1版本释出.有如下更新: 1.添加单一运行模式。 2.在打开的情况下接收命令行发送来的网址。 3.添加pps支持(试验功能)
新版本需要配合gmlive-extra-0.2 来使用.

详细说来，单一运行模式即全局只能打开一个gmlive实例。不过已打开的gmlive可以接收命令行发送来的网址。比如在终端输入 gmlive mms://xxxx.asf 这样命令，而此时已经有一个gmlive在运行中，则运行中的gmlive则可以播放发送而来的这个mms://xxx.asf网址。由此可作的引申功能是让gmlive和firefox作关联，打开mms:// ,synacast:// ,pps:// 这样的协议时可以直接调用gmlive来播放。

另外,pps的支持在试验中。并且它的二进制程序要求root权限，请确保您明白这一点的意义，及自行承担风险。详细的说明会在gmlive-extra-0.2中。

gmlive 0.22.1下载地址 http://gmlive.googlecode.com/files/gmlive-0.22.1.tar.bz2

汇报一下

| Comments

gmlive的pps支持工作第一步完成了。喜欢尝鲜的同学可以去
http://code.google.com/p/gmlive/downloads/list
下载。新版本的gmlive是0.22.1，需要配合gmlive-extra-0.2使用。(迟点放正文)

从pps发布linux客户端到现在，过了一星期又一天。其中也已经出现了许多pps的客户端程序。
gmlive也加入其中，当然或许会有不足，会有bug，但相应会越改越好。至此，gmlive已经能支持多达三种P2P网络视频流。

老调再重调一次。gmlive是个网络视频外壳程序。并不直接提供各视频的内容。而对于支持各种网络视频流的程序，是通过gmlive-extra中从网上下载的各种流二进制程序实现的。

比如sopcast是它公司公开发布的sp-sc-auth 命令行程序。
pplive的支持是某人移植mid中的xpplive命令行程序。
ppstream的支持是无名氏通过ppstream网站上发布的sample.zip及库写成的xpps 命令行程序。
以上都是通过网络下载而来，经gmlive包装调用，以方便用户。

警告:pps的程序需要使用到root用户，尽管可以通过设置suid位解决，但仍有风险。请警惕使用。

Ppstream for Linux

| Comments

星期六晚上上网，看见ubuntu中文论坛正讨论一件事，就是ppstream for linux发布了。这真的是非常奇特的一件事。当时就下载试用了，但官方放出的程序只能是个半成品，试验版。终端运行，有限的频道，改版的mplayer,极奇怪异的是要求使用root权限才能运行。但随即发现在pps的下载页面，不单止发布了那个”测试”的程序，还包含了一个叫 sample的压缩包。

这个压缩包里的两个文件详细解说了pps库的使用方式。将里面的测试源码变成可用的程序只有一步之遥。这还真是让人摸不着头脑。压缩包里完全没有提及作者及版权信息。但凭着这个源码及发布的pps库，一个成型成熟的pps for linux程序无疑将很快被做出来。不要小瞧开源力量，才一天时间，已经有人用tcl实现了频道列表的获取与播放。而pps获取的这个列表无疑比移植来的pplive的列表更丰富。pps官方又是打着什么样的算盘呢。各种山寨pps程序起来后，无一不是连接着pps提供的没有任何说明的pps库，而这些程序也不大可能嵌入pps官方的广告。只是先利用开源的力量占领Linux下的终端吗。无论如何，现今Linux下的网络电视已然有了三大解决方案（sopcast，pplive, ppstream). 各路软件厂商都纷纷将目光投向了Linux,这是好事。向着好的方面发展着。

一个问题，gmlive会支持pps吗。记得gmlive的目标吗，成为一款通用型流媒体播放器外壳。支持多种网络视频正是它的特色。而gmlive也正等着三大网络电视程序聚首的这一天。当然，gmlive并不会对这些程序做些什么，它只是帮助用户方便地管理及调用各种网络资源而已。那么，什么时候呢，这个就没上次做pplive支持那么快了。一来我们有点忙了，二来要做的工作也相应增加了。但绝不会太久，敬请期待。

毒瘾,网瘾,股瘾

| Comments

昨天我看了《窃听风云》，其中有这么一幕。平时冷静理智的小祥，在股票交易所里，和所有股民一样，为股票忧，为股票喜。为股票飙升而狂兴奋态。

而在这时我在想，成人世界能定义出网瘾这种名词，为啥没定义出股瘾。这两种瘾有啥区别。
毒瘾让人身陷毒品，沉迷其中，不问世事，一离开毒品就身体反应，情绪受控，戒断反应。
网瘾让人身陷网络，沉迷其中，不问世事，一离开网络就心绪不宁，烦燥不安。
股瘾让人身陷股市，沉迷其中，不问世事，一离开股市就

毒瘾是看见人家吸（毒），自己很想吸。
网瘾是看见人家上（网），自己也想上。
股瘾是听见人家聊（股市），自己也要上前谈两句（当前股坛大势）。

其他人无法理解毒瘾患者的世界，吸毒达到的高潮是多么爽，多么忘记一切烦恼的。
其他人无法理解网瘾患者的世界，在网络中多么舒畅，指定江山，激扬文字。
其他人无法理解股瘾患者的世界，在股市随着指数的升高而肾上腺素提高，获取了金钱并达到人生的高潮。

毒瘾的人喜欢拉别人一起吸毒，有乐一起享，一起high到最高点，它抽象了我们的肉体。
网瘾的人喜欢拉别人一起上网，明明近在咫尺的距离，但在网络中才会更亲密无间，无时无地在线，它抽象了我们的精神。
股瘾的人喜欢拉别人一起炒股，有钱一起赚，最重要一起拉高指数，无时无刻不在想买进卖出，所有成败在此一举，它抽象了我们的人生。

毒瘾的人经常会说戒，不再吸，他知道这是不好的。有时也请求别人帮忙，但过不了又恢复吸，瘾大难掉啊。
网瘾的人也经常会说戒，这个戒是指控制上网时间，因为他知道长时间上网是不好的，有时也请别人监督，但上着上着，又忘记时间了，瘾大难掉啊。
股瘾的人也经常会说颧，这个戒是指控制炒股的次数及成本，因为他知道大比投入是不好的，有时也请别人监督，但看着股市的势，他又觉得肯定会升，肯定会涨，肯定会赚，又投入进去了，瘾大难掉啊。

毒瘾的人没钱吸毒了，想尽一切办法赚钱，偷拐抢骗等一切违法的事，总之能搞到钱买毒品。当然这之前会向周围的亲戚朋友借钱，直到人见人憎。
网瘾的人没钱上网了，想尽一切办法赚钱，偷拐抢骗等一切违法的事，总之能搞到钱上网。当然这之前会向周围的亲戚朋友借钱，直到人见人憎。
股瘾的人没钱炒股了，倒是很少做偷拐抢骗等一切违法的事，因为想尽这些办法赚到的钱，不如股票里升一个小数点的多，股瘾的人不会没钱炒股的，基本上属于负资产炒股的，所以他们要赚钱，实际上还是要在股市里赚，直至套牢，结束生命吧。当然这之前会向周围的亲戚朋友借钱，直到人见人憎。

我这里只是随便列举了一些，过尤不及，任何正确的事，都会有人将之做尽及至做错，好事也能变成坏事。这取决于人。既然能定义出网瘾，也就可以定义出股瘾。也完全可以让那些患有股瘾的成人去让人电一电。而为啥股瘾不出现而网瘾出现呢，一是全世界只注意到青少年问题，而忽略成年人问题。二，这是成年人掌权的世界，因为掌权的人不可能出问题，要出问题也是个体问题，不可能是群体性问题，因此世界上的群体性问题对于成年人的世界来说，就是青少年问题和老年人问题。因此我们又看到一种权力自满及不会自审的现象。

在ubuntu中的ppa仓库里安装GMChess（转）

| Comments

使用Ubuntu的同学们现在可以使用添加源的方式来安装gmchess了。

首先呢，添加PPA仓库的key 

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 04F95913
然后在新立得软件包管理器里选 设置->软件库->第三方软件库
点击添加，按照系统版本输入内容

9.10用户：
代码: 

deb http://ppa.launchpad.net/gmchess/ppa/ubuntu karmic main
deb-src http://ppa.launchpad.net/gmchess/ppa/ubuntu karmic main
9.04用户：
代码: 
deb http://ppa.launchpad.net/gmchess/ppa/ubuntu jaunty main
deb-src http://ppa.launchpad.net/gmchess/ppa/ubuntu jaunty main
然后点击“添加源”确认，最后按照提示刷新软件包列表。

完成后即可通过新立得搜索 gmchess 安装，或使用以下命令安装：
代码: 

sudo aptitude install gmchess
以上内容引用自ubuntu的cnfavor的贴

监控本本的cpu和硬盘温度

| Comments

使用笔记本的话，基本上对温度都比较敏感的。所以在桌面上的工具栏中总有一项是显示温度的。
要监控温度需要的软件. lm_sensor , sensor_applet, hddtemp.

用你的包管理器装完以上软件后。在gnome工具栏右键添加”添加到面板”，选中 “Hardware Sensors Monitor” 即可。
基本上有cpu的温度了。而要监控硬盘的温度则需要运行hddtemp，将其设定为后台服务。之后在传感器属性里就会出现hddtemp的选项了。
