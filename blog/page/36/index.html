老爸的简单历史

| Comments

远在异乡，却突然想起爸爸。

老爸家里兄弟姐妹众多，排行第三。家里养不起，于是老爸的爸爸把老爸送给别人养，至十三岁回归。读完小学后就出来打工。老爸因此从小就很独立，但性格就比较静默。

二十岁左右的时候经人介绍对象，就是我老妈。说，我家里有鸡，结婚有肉吃。于是成了。在村里排了三十几桌。有这钱，是因为之前给一学校做桌子，赚了三千多。呃，那真是比巨款。这时候，老爸是木匠。

结了婚也得有个地方住啊。于是乎自己拉砖头，在村里建了一个六室一厅。有了房，就可以养儿育女了。于是开始有了我姐姐。再有我的时候，老爸为了生计，独自投入到市场经济的大潮中。做啥生意不知道。自我会说话，就问爸爸在哪里。老妈回答：爸爸在湛江赚钱。

后来老爸跑得更远。在江西开了个汽车装潢店。把全家接了过去。一家人挤在店里。即是店，也是家。从玻璃贴纸，到座位坐垫，再到装空调兼加水，几乎一个人干完。九零年那时就在那买了一套房。后来因为拆迁，拿了几千块补偿，老爸无言以对，又租了一个店面重新开始。这回把赚到的钱都汇回老家盖房子。其后几年收了一个徒弟继续干。但随着所谓改革的深入，也越来越多人投入这行，生意渐渐难做。而他的儿子正处于成长叛逆期，学会了逃学，混游戏厅，屡教不改。期间店面也遇到一些流氓敲诈。正所谓内忧外患。于是决定效法孟母三迁，把他的儿子，也就是鄙人，带回老家。于是把一家人迁了回来，自己又独自奋斗了两年。人生有得有失，顺境逆境。老爸年纪见长，看透世情，决定提前归隐，回来一家团聚。从此在家里开个小店过日子。

但每每见他，却不知说些什么。父亲性格如此，儿子也受此遗传吧。平时电话只互嘱要保平安，情浓于心中。

vim技巧之取词翻译III

| Comments

以前写过vim里用sdcv取词的一系列文章。

按一个快捷键能调出窗口来显示sdcv的翻译结果，但它不能方便地调用关闭。也就是用同一个快捷键开关窗口。如今此vim脚本有了进化，加了如此功能。

全部脚本如下： 

let s:w_mywin=0

function! Mydict()
if s:w_mywin
let s:w_mywin=0
set nu!
close
file
return
endif
let s:w_mywin=1
let expl=system(‘sdcv -n ’ .
\ expand(””))
windo if
\ expand(“%”)==”diCt.css” |
\ q!|endif
botright vertical 25split diCt.css
setlocal buftype=nofile bufhidden=hide noswapfile
set nu!
1s/^/\=expl/
1
endfunction
nmap :call Mydict() 将之保存为sdcv.vim,放到~/.vim/plugin即可。
详细信息参考: vim技巧之取词翻译II VIM技巧：VIM里快速翻译单词

再次无题

| Comments

一段时间都没有写博客了。甚至有些遗忘这里。感觉这里一直是我理想的阵地。超越现实的精神世界。

而前一阵跟老婆聊天，她说我转变了。从以前的理想主义转变成了现实主义。因为她以前问我在干什么的时候，我似乎都在写程序，而现在问我在干什么，我答不上来。似乎不是看电影就是上网。上网上网，无尽的上网。以前的我迷失在北京了吗。

网络上有太多的新闻，现实的消息，然而不能增进我的思考，也只能加剧人的愤怒。

为了打发地铁上的时间，我买了一个电子书设备，就是电子墨水的那种。看txt还行，pdf刷新有点问题。在看一本禁书，《远×华×案×内幕》。里面讲的官场黑暗，全党溃败。当然里面人说的话不能尽信。也只能当小说看。因为你当现实看了，那会扼杀希望的。

博客里是不应该发这些不和谐的言论的，然而在twitter里就可以。在twitter出现的时候，总认为博客才是正道。现在看起来，twitter才是能自由畅言的地方。然而twitter默认是在墙外的。我们这么争取言论自由，不正是因为这是我们最缺少的东西嘛。而党国一再强调要和谐，不正是因为这个国家一度地不和谐嘛，而常摆在嘴边的以法治国，不正是说明国家经常就不这么干嘛。经常说反腐，不正因为腐败广泛而深入嘛。

为什么中国人命运就如此坎坷呢。我发这些唠叨又有什么用呢。

24小时第八季完结

| Comments

一直都在追的美剧。《24》，主要讲米国ctu探员jack在一天24小时内拯救美国，粉碎恐怖分子的事情。将剧情压缩至一天内，张力十足。这八天中，多次破解核弹危机，病毒危机，以及米国总统阴谋。甚至连天朝大使馆也没放过，呵呵。

以下为剧透：

第八季一直围绕着一个所谓和平协议开展的。看到23集的时候，我真替泰勒总统感到委曲。事实其实是，泰勒总统一直想打麻将，好不容易约齐了两个雀友。irk共和国总统和俄国总理。irk总统也是雀友，但问题就在于俄国总理不是。他非常喜欢打麻将。但总不好当面拒绝米国吧。人家说到底也是世界第一强国，总不能不给面子吧。于是搞个阴谋把irk总统给干掉了，这回三缺一，打不成了吧，正乐着。谁知道irk总统的老婆也是个麻将高手，在泰勒的劝说下顶了她老公的角色。好嘛，人齐了，准备开台吧。却在这时候俄国总理杀死irk总统的事让新接任的她老婆知道了。新irk总统就不愿意和杀夫仇人一起打牌了。泰勒总统只能软硬兼施，好不容易让这两人坐在麻将桌上。可是看着桌上的人，不就打个牌嘛，左一个哄，右一个骗，威逼利诱全用上了。泰勒总统想着想着，我咋那么二哩。这牌打起来没意思。于是半途自个走了。

当然24小时结束了，我会想念小强的。当然也想念chole。小强最后一句话，也挺感动的，说没想到chole会是一直都以来支持他的人。在24小时里，除了小强不死这个常量外，任何人都可能随时牺牲，任何探员都有可能内奸。另一个常量就是chole永远是小强坚持的后盾。他们里应外合，合作无间，所向无敌。

24小时不会再有第九季了，我都怀疑，以后能否再看到如此紧张刺激的美剧了。i will alway miss you.
