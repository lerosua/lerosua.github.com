Talkmm 0.10版发布

| Comments

talkmm是个什么东西哩，话说我还没正式介绍过哩。

按某人的解释就是gtalk for linux。嗯，差不多吧。就是一个用gtkmm作界面，libjingle作后台的gtalk客户端。目前只支持gmail帐户登录，的确是可以说是linux版的gtalk了。它支持与gtalk(win版)传文件，语音通话。是的，这些功能都支持。

当然，就其方面就比较简陋了。但，这毕竟是第一版嘛，测试用。相信它会越来越好的。

talkmm编译起来相当的麻烦，没经验的用户不要轻易尝试了。相应不久就会有对应的打包出现。而本人也只能提供一个Arch下编译的帮助（见前篇文章）指南。在下一版中将尽力简化这个问题。

另外，talkmm将会在下个月的google开发者大会中露下脸哩 :-)

Arch下编译talkmm

| Comments

Arch下talkmm编译指南

1.编译前准备的包
语音支持方面的包 

yaourt -S speex ilbc linphone
界面支持方面的包 
yaourt -S gtkmm libsexymm
2.下载源码包 
wget http://talkmm.googlecode.com/files/libjingle-0.4.1.tar.bz2
wget http://talkmm.googlecode.com/files/talkmm-0.10.tar.bz2
3. 解压libjingle 
tar xvfj libjingle-0.4.1.tar.bz2
cd libjingle-0.4.1
./configure
make
cd ..
4. 解压talkmm 
tar xvfj talkmm-0.10.tar.bz2
cd talkmm-0.10
5. 连接libjingle目录 
ln -s ../libjingle-0.4.1/talk src/talk
注意这是在talkmm-0.10目录里做的，并且libjingle-0.4.1目录就在talkmm-0.10同级目录中。
这是为了talkmm里的源码能连接到libjingle里生成的静态库.

6.编译talkmm 

./autogen.sh
./configure
make
sudo make install
PS. Arch中gcc 4.4.0-3 版本在编译时似乎有个bug。目前只能跳过这个gcc的版本来编译.

如要指定ilbc位置,就以下面方式 

./configure --with-ilbc=/usr/local
更正：前面说是4.4.0-3编译有bug，是我记错了。其实是4.4.0-2的bug，在线程同步方面。当时编译不过我就降级了，没想到过没多久gcc又升级并fix了这个bug。大家找的那个patch打上也无坏。它也修正了一些问题，我也已经把它加入svn中。

上半年的美剧终结

| Comments

2009年上半年我追的美剧大部分在这星期及之前结束了。然后又是漫长的等待。等待着下一季的来临。

先是前几星期就完结的《heroes》第三季。第三季分了两章。前一章讲恶人出道，放出来的恶人没怎么恶就被syler干掉了。然后syler又莫名其妙地反水。boss是peter老爸，吸收了N多超能力，看上去很强大。结果被干掉不用一分钟。我总是期待能有场大战，但超能力在里面全被浪费了。意识到超能力编的过火后，后一章两最强超能力的人都被削弱了。peter的万能吸尘器变成了只能吸一种，hiro的时空控制时只能停住时间，不再能穿梭过去未来了。但这一章也更郁闷，被政府特工追杀。syler又变得更加强大。最后终于又大战一场，啥场面都没有，syler又被搁倒。perter他哥又继续每季必亡的命运。总结来说，人物性格反复无常。

《越狱》其实上是贵国最热的美剧了。迎来了终结季。这名字本来就只好放一季的。其它季根本与主题无关。而为此，让主角不得不在第三季再越一次。这都算得上第一季火爆的余威吧。第四季的剧情就太扯了，一开始公司的能量何其大，最后又何其不堪一击。那个傻傻的高科技设备只不过一张嵌入式开发板，存资料可以用sd卡，u盘嘛。非要搞个板干嘛哩，唬弄不懂的同志吧。最后又莫名跑出一个组织帮忙搞掉公司。我说那板子存的是未来科技又不是犯罪证据。没了板子就被抓，这啥狗血逻辑嘛。当然，七月份还有个相关的片看看。最后缅怀《越狱》就在那时了。

《Lost》是悬疑剧的经典啊。不过这季的表现是有点中规中矩了。一伙人回到了三十年前。俺可怜的sawyer的平静幸福生活被打破了。最后的结局又放出了更多的问题，那个活着的john到底是谁，jack一干人等在氢弹爆炸后会怎样。这多数是回到现实的时间中来的了。迷底也只能期待下一季了。

《24》今晚刚看完的。如所料，tony是玩终极无间的。jack奔波了这么多个24小时，终于有觉好睡了。本可以一睡不起的，怎知kim又要治疗他。无论这手术处于多么实验阶段，治好的机率多么小，我们都知道，jack在下一季又会东奔西跑，杀人投弹并严刑逼供。谁叫他叫小强哩。下一季，还是让ctu回来吧，挺相信这组织的。

星期六志

| Comments

一直以来对所谓微博客都不太感冒。直至昨晚关注了一下佐拉在列车上与乘警的冲突。我才对这种媒体的即时性，以及把互联网带进生活有了点深刻的体会。

在生活中遇到一些执法人员的不礼貌对待几乎是常有的事。而遇到这类事情，我们也只有忍气呑声，独自承受。也许事后想就此写篇博客声讨之。但当时心境已过，又本着多一事不如少一事的原则，淡而忘之。久而久之，几乎就是默认于放弃了保护自己尊严的权利。而已有之权利被侵食的就欲加厉害。而有了twitter这类服务，几乎是即时传递身边的信息。遇到不公不正也可马上声张权利，激起大家保护权益的信心。

当然必备之物就是一部互联网手机了。总不好到处都带着个笨重的笔记本吧。

才明白，笔记本之便捷性也已经落后了。电脑这类设备，是把我们带进互联网。因为我们只有坐在它面前才能畅游网络，指点江山。于是产生宅之一族。而手机这类设备，才是把互联网带进生活。因为它可以跟着我们到处游走，并且随时随地连接上网络。

我的生活已然是离不开互联网的了。一日不上网，则烦恼不安。但整天宅在电脑前也实在不是好事。看来一部互联网手机将会是必备之选。买手机的欲望更加强烈了。

饥荒的故事

| Comments

最近晚上都在追香港的一部电视剧，《巾帼枭雄》。主要是讲江苏那里一个卖米商家族的事情。

其中那个四奶奶对蒋乔（卖米商）说，每逢大饥荒，总有七分是天灾，三分是人祸。而我们作为江苏最大的米商，有责任减轻这种人祸。而第一集中就讲了这个米商捐了自己三分之一家产来赈灾。

当然电视剧就是电视剧，没必要深入研究仅凭一家富豪是否能减轻天灾中的人祸。而且也没有追咎人祸产生的来源。实质上在灾难中，人祸导致的伤害不可估量，却也往往被掩盖在天灾之下。

一旦发生饥荒，那里的县令首先会瞒报消息，限制饥民的流动，因为县令的升迁是要看政绩的，自己的治下发生了饿死人的事件，那当然影响自己的仕途。所以第一件事就是封锁消息。平时那些打更的啊，说书的啊，都命令他们禁声，不许到处说。灾民们没有出路，只能上京告状去。县令出去了衙差用棍打啊，用铁链锁人啊，以此镇压一下。但实在是太多人了。于是县令亲自出动，跪在各位乡亲面前。各位乡亲啊，眼下正老佛爷大寿哩，大家别去京城添乱啊，我知道大家都很苦，但大家忍着点，别出去丢了俺大清国的形象啊。一口一个爱国情怀把灾民感动回去。

当然饥荒的规模开始扩大了，这回也实在瞒不住了，上头派了钦差大臣下来调查。并且带了一百万两赈灾银。县令马上布置人手开放粥场，并带钦差大臣视察环境，当然被视察的人都是事前安排好了的。于是这般走过场后，钦差大臣与县令一人一半私分了赈灾银两后回京复命了。县令得了赈灾银后拿出一部分打通各级的关系。并广布消息，天子圣明，广开粥场，赈灾效果非常好，百里之内再无饥民。说书的拿到钱后，也到处宣扬此番政绩。放眼下去，四海之内，歌舞昇平。

一年后，劫后的灾民集体悼念在这次饥荒中的死去的亲人。却被县令指责是非法集会，聚众闹事。而在本县志中，记载着上次饥荒朝廷及时开仓赈灾，官民一家亲的和谐景象。还有一些秀才画了一幅《县令送米图》送给县令，还题诗，有此父母官，做鬼也幸福。

此县令凭着此次表现良好的政绩，以清官，为民执政等称号升了一级。

嗯，你看懂我说的故事了吗？其实我是在说开头的那部电视剧啊。别想歪了。

物理学知识

| Comments

虽然在天朝，人民是当家做主的，而各位公仆也是大公无私的。但“中国人是需要被管”的，所以公仆与砖家们经常出来教育人民。

比如教育人民学习ps技术的“虎照门”。

比如教育人民锻练身体的“俯卧撑”。

比如教育人民学习化学知识的“三聚氰胺”。

而在前几天，相关部门又出来教育人民学习物理学知识了。一辆车，在公路上以70码的速度奔跑，撞上一个人，请问最多能把人撞到多高？答案是五米高。

或许你怎么想也想不通。这不违反物理学定律吗？其实是这样，因为俺们身在天朝，天朝的国情特殊啊 ，俺这里的重力系数比别的地方都轻。没见电视上到处都是飞檐走壁的人嘛，不见得国外的人是这样的，这充分证明了俺天朝的重力系数小的原因，因为70码的车速完全是有可能把人撞到五米高的。大家在用物理学证明这一事件的时候请充分考虑天朝国情的重力系数。虽说同样是物理学知识，但千万别随便和国际接轨。

俺骑着草泥马，以70码的速度远离马拉戈壁草原，因为那里已经被河蟹所占领。俺大喝一声，”卧草”，草泥马跳起五米高，跃过了马拉戈壁草原的地界碑，却发现草原以外的世界，仍然布满了一遍河蟹。
