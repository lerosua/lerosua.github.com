《立春》

| Comments

一开始有些反感王彩玲。因为她的第一句话就是北京正调我呢。在后来的场合她

也主动地提到过几次。这个人的确有些与众不同，并不是与众不同，而是与世不

容。与之个城市格格不入。这样的状况主要还是她个人性格造成的。她有理想，

想去巴黎歌剧院唱歌。而且坚持到三十多岁。坚持到别人早就把梦想抛弃，踏

踏实实日子的时候。在这个时候，她也就成了异类了。
其实我是属于理解她的。因为我还没过放弃理想的年龄。我不想这样平凡地过，

不想碌碌无为。不想在这个小城市里发生爱情。这些想法，我都能理解。人都

是有虚荣的。所以后来我理解了一开始的王彩玲。之前的反感一扫而空，转而

关注起这个人物。虽然说她立志不在这个城市里发生爱情。但是春天里青蛙求

偶的叫声还是一如既往地叫，别误会，电影里没有这种叫声。替而代之的是隔

壁房两小夫妻的做爱声。声声入耳，还是把王彩玲带入了春天的境界。她终于

也低抗不往内心的萌动，冲动了一回。王彩玲并不是不想和别人沟通。只是她

一直找不到可以沟通的人。直到遇到黄四宝，才算是一个可以说说话的人。可

以借书给他看，可以问他关于俄罗斯三姐妹的事，才会甘心让她画女体像。才

会有找到知音的感觉，才会感觉到爱的冲动。而这一切，都因为她一直太过于孤
独。

不甘于世俗是可贵的，也是可悲的。这与时代无关。

所以在经历了种种挫折之后，王彩玲终于于世俗妥协了。她去征婚，去领养孩子。

为孩子取名王小凡，终于领悟了平凡的生活也是一种幸福。

虽经历种种，但王彩玲曾经坚持了，曾经高歌了。我想，也是无悔了。

今天是好日子

| Comments

近几天下了大暴雨。岗顶那都成了泽国了。新闻是这样说的。

我没有去看过。广州的城市建设真他妈差啊，一下雨就堵塞管道。

还申啥文明城市呢。我每天上班都经过一条河，说是河，其实是臭水沟，

黑龙江。什么垃圾都有，一阵风吹过，方圆百里都臭。建设和谐社会，

拜托把环境搞好点先吧。

不说了。本来一醒来，阳光挺美好的。心情也该是美好的。重新开了一个博客。

希望能长长久久地写下去。

Icalk 0.01-4截图

| Comments

可以发送小于100K内的图片。png和gif格式的。本来是可以jpg的，不过打开文件的那个过滤器有问题，
过滤不了jpg的。而且那个任何文件的过滤也是更有问题，一选就没有文件出现了。
这个功能用的是带内数据传输的。也就是把图片转成base64，传输出去。而后面版本的gloox把带内数据这
功能集成到文件传输里去了。也许下一版本就没有这功能啊。真奇怪。

Release Icalk 0.01-4

| Comments

发布一个 0.01-4的版本。编译安装前请升级gloox为 0.9.6.1-c1。

主要改进：
修正以前的一些BUG。
添加传递小数据量的图片功能（大小限制为100k,图片越大

，收到的时间越迟.
依赖gloox-0.9.6.1-rc)
添加消息签名超时。（在签名管理窗口里选上MsgTimeout选项。签名会五分钟
变换一次，为随机读取签名文件里的签名)

仍然热切希望有人能加入一起开发~


source code:

http://icalk.googlecode.com/files/icalk-0.01-4.tar.gz
http://icalk.googlecode.com/files/gloox-0.9.6.1-c1.tar.gz

ubuntu 7.10 deb (deb是我用checkinstall做的，搞不到依赖关系，所以请先安装libgloox再装icalk)

http://icalk.googlecode.com/files/libgloox_0.9.6.1-c1_i386.deb
http://icalk.googlecode.com/files/icalk_0.01-4_i386.deb

English:

i had release icalk 0.01-4. build it before you update gloox to
0.9.6.1-c1

feature:
fix some bug
it can send a picture which size under 100k now.(testing , and only
between two icalk client)

“StatusMsg manager” window)
