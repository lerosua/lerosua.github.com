键盘快速输入

| Comments

在Linux命令行下一些常用的快捷键,能提高输入速度!
例如用ctrl+j来代替回车,ctrl+h来代替backspace,
减少手在键盘上的移动,极大地提高输入速度.另,这在vim
中也有一部分是可用的.

ctrl+u 删除光标以前的所有字符

ctrl+d 删除光标以前的一个字符,相当于delete

ctrl+h 删除光标以后的一个字符,相当于backspace

ctrl+y 粘贴之前用ctrl+u/k 所剪切的文字

ctrl+t 调换光标前两个字符的次序

ctrl+a 移动光标到最前面

ctrl+e 移动光标到最后面

ctrl+p 上一个命令

ctrl+n 下一个命令

ctrl+s 锁定输入

ctrl+q 解除锁定

ctrl+f 移动光标到后一个字符

ctrl+b 移动光标到前一个字符

ctrl+x 标记一个位置

ctrl+l 清除画面

ctrl+c 结束命令执行

ctrl+j 执行命令，相当于回车

ctrl+m 同上

tab 命令补齐

另，alt + . 是上次命令的最后一个参数。 alt + 1 ,alt + . 是上次命令的第一个参数。详细的请参考bashline.

因为常用，所以记下来！

用W3M终端下上网指南

| Comments

这里指的终端是指不启动X的那个终端！
首先保证你的内核支持framebuffer,因为有这个可以显示图片。

安装w3m, w3m-img(这个可以显示图片） 

sudo apt-get install w3m w3m-img -y
为了显示中文，你得安装zhcon 
sudo apt-get install zhcon -y
那么现在，可以转到终端下面了！

1. 先启动zhcon吧！一般在终端下面是不能支持中文的。

2. 如果你的locale不是gbk的话还是会乱码的。所以运行 export LC_ALL=zh_CN.GBK

3. 启动 w3m 吧。启动方法: w3m www.baidu.com .可以看到熟习的网页了吧！

但是不会用w3m怎么办？ 不要紧，下面马上就传授几招必杀！

常用操作

与vim相似，h,j,k,l 可以分别用来做移动键,分别是左，下，上，右。如果你熟习vim操作的话这会是很方便的功能。 < 和 > 用来左右滚屏。

按q就会提示你退出，所以千万别乱按q了！

想输入怎么办？ 在需要输入的文本框内按回车，下面就会出现TEXT: ，这时你可以输入你要搜索的文字。再按回车就是返回给文本框。这时再把光标移到”百度搜索”的那个按键，回车，就可以开始搜索了！

使用 U 来重新输入需要打开的网址。

B 返回前一个页面。

多标签操作！

你可以使用 T 按键来打开一个新标签。

怎样在多个标签内切换呢？ 使用 { 和 }就可以了！

使用 ESC-t 的话会打开标签的菜单让你选择，功能类似系统中的alt-tab功能。

C-q用于关于当前标签页。

书签

C-a 添加书签

C-b 查看书签

查找

/ 向后查找当前页面

? 向前查找当前页面

n 查找下一个已查找过的关键字

N 向前查找已查找过的关键字

帮助

H 以上的选项在这里都可以看到！

写得比较乱，但一般使用应该是没问题的了！当然一般是不会有人想到用w3m来上网的吧！不过，哪天你X启动不了，试试它吧！

从公司恍恍惚惚地出来,”菠萝王”约她女朋友出来逛街,所以偶只好一个人回学校了.看看时间是9:05分,想想,昨天夜里九点多的时候打电话给”石头”,不知什么原因,她并没有接我电话.而这个时候,她应该已经起床了,因为她跟我说过,她一般是8:30起床的.由于,她可能会随时会回家,而且偶有一张光盘要送给她当做毕业礼物吧.所以就拿起电话拔了她的号码,郁闷的”嘟”声以后就是中国移动的提示音,你拔打的电话无人接听.最讨厌这个提示音了.看看手机,在人来人往的大街上,傻傻地笑了一下.可能是她在忙别的吧,没有注意到电话在响吧;或者是睡得昏死过去了,不知道有来电.假想了N种可能,也没有想打第二次的念头了,那是骗人的,其实是很想再打一次,那样的话,显得自己很无趣,很”孙子”,用陈大导演的话,人不能无耻到这种地步. 有时,上天总会跟你开一个小小玩笑.在我认为有公交车(番禺的中巴并没有像广州那样标明着有什么线车经过此站的)经过这站的站点上等了差不多一个小时,可就是没有见到一辆车经过,就连开往反方向的也没有.我开始怀疑自己的记忆力起来,是不是我走错了乘车的地方,可事实告诉我,此站确实有一线车回去偶学校的.因为,那是世昌宾馆,那里有307,那是沙湾古坝西专线经过的站点啊. 恍惚之间,怀疑起自己所有的东西. 心情急燥不安,脑袋更是乱哄哄的,实在是没有心情再等待下去了,改变主意乘座6B线车回到沙湾文化中心,然后再乘车回学校.本来可以花2.5元的车费就可以回到学校,现在却要花3.5元车费才回得到学校.谁叫上帝总是喜欢跟我开这样的玩笑呢? 在公交车上,昏睡了一会儿,似乎还做了一个梦,或者说是让以前的记忆清淅了一些.隐约之间,想起以前跟”石头”仅有一次乘坐公交车的回忆,似乎是发生在昨天的情形. 有时,记忆是一种骗人的东西. 回到学校,已经是十一点了. 懒散地走着,看看那守门的保安,之前就说过,那些鸟保安,每每有漂亮女生经过,都看用上他们那特有的”色眼”盯着女生不放,意淫好一会儿.还有,跟《天下无贼》里的那个保安一个样儿,见到好车就敬礼.不一样的是,《天下无贼》里的那个保安敬敬礼时,身体站得笔直,而偶校的是站得还没有偶坐得要笔直. 鸟保安从视线消失后就是偶学校所谓的校训.都不明白,什么事儿都要刻在所谓的石头上,绪如,碑记碑文之类的．什么记到最后还不是墓记． 还好，还没有到放学时间，所以校园里空荡荡的，只有零散的毕业班生搬东西离校，在他们眼里离开学校,这个时候是多么伤感啊,

如果有可能的话,最好用摄像机来个大大的特写,至少要比A片中特写”那一片森林的地方”还要特写. 为了落个清静，靠着湖边的小路走．湖里的荷花，红的，白的，紫的．从来都没有感慨过＂出于泥而不染，濯清涟而不妖＂，而感慨的是荷花荷花啊，为什么你要在这一湖臭水里生长啊，难道你就不能选择干净的水吗？难道你就要这样做贱自己吗？离开那一湖死水吧，清风激不起半点涟漪．为了不做贱自己，离开那一湖臭水吧．清水永远会欢迎着你的，我敢保证．可你就是喜欢这样，你真是他妈的做贱自己． 关于湖的记忆，可以追溯至大一．傍晚绕着西湖、南湖（北湖并不适合跑步，偶学校没有东湖）边跑步，目的似乎不是为了跑步，而是为了说不定对面也迎来一个ＭＭ，然后……真是他妈妈的天才，太会想像啦。事实证明，来跑步的都是猪扒，因为要减胖啊，所以记忆当中只跑过一次步。北湖湖边长椅多一些，所以那些情人们似乎很喜欢去这样的鸟地方。有一次，从自修室回来，已经是十二点多，冷得要命，路灯也关了。黑暗之中却发现Ｎ对情侣们，在我看来他们只是身体靠近一些，并没有什么。可是鬼知道他们在干些什么啊，说不定那手已经把那乳罩扯得不见踪影，那片地方已经泛滥成灾，还有那坚实如棒，或者说从没有如棒过的家伙．至于西湖，唯一一次的就是跟那鸟师妹约于西湖．谈论的却是漫无边际的话，跟风花雪月之事一点儿关系都没有，真他妈的牛Ｂ．

听水王阿耀说，西湖边用过的ＴＴ可是多的要命，这也难怪了，西湖树多，而且偏僻，更要命的那草地坐上去非常舒服，更不说趟上去干几下啦．真后悔当初没有带ＴＴ去，干上十回八回，那个后悔啊，伤心啊，人生最大的痛苦莫过如此． 回到宿舍，已经是午饭的时候了，众兄弟提议去知己餐厅聚餐，觉得跟大伙吃吃散伙饭也好．出发之前，越想越郁闷，还在为＂石头＂不接我电话而郁闷中．我是小人啊，我无耻啊．更无耻的是，用＂鸟蛋＂的手机拔＂石头＂的电话，跟我想像中的一样，我真她妈的是预测师．一秒之内，电话那端马上传来＂喂＂的一声，像这种小人早就想好应对策略了．＂喂，小花啊，你在那里啊？＂＂你打错电话了吧．＂＂哦，很抱歉．＂这一次却是没有笑得起来啊，真想从六楼跳下去．自己的分量还不如那几毛钱的分量，自己的号码人家是从来不接的，因为那样会浪费人家的话费啊，所以说，老犁啊老犁，你说的全是废话，老犁你算老几啊，你还不如我那几毛话费呢． ＂石头＂啊＂石头＂，首先，我承认我是小人，我比小人还小人，行了吧．可石头你呢，难道你忘了，不辞劳苦地从华师带书回给你．更要命的是，在书店里不断地拈量你的接受程度，选深的不行，那会打击你的学习热情，选简单的不行，那会让你飘飘然，不经典的图书不选．在华师的书店里一呆就是三个小时．从书店里出来，饿得发毛，华师西门的小吃可谓云集，当最终还是选了兰州拉面馆吃面，不为别的，就为了那样能省下几块啊．你可以说我是抠门之人，可是，我对朋友，我对你，抠门过吗？没有．那书给你的时候我连一个钱儿都没有提过，不是因为我有钱也不是因为我要追你之类云云的，而是冲着你热爱计算机，是冲着你跟你交个朋友而已啊．连火星人都知道你男朋友是那苏卫标的家伙，在我看来（也是客观的），整天逃课躲在宿舍里玩打机，专业素养更不用说了，人长得也很猪头（这是主观的）．当然，我不想跟他相提并论
